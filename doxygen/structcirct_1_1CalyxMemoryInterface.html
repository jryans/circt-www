Detailed Description

The various lowering passes are agnostic wrt.

whether working with a calyx::MemoryOp (internally allocated memory) or external memory (through CalyxMemoryPort). This is achieved through the following CalyxMemoryInterface for accessing either a calyx::MemoryOp or a CalyxMemoryPorts struct.

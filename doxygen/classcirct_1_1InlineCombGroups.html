Filter groups which are not part of the control schedule.

Maintain a set of the groups which we've inlined so far. The group itself is implicitly inlined.

Starting from the matched originGroup, we traverse use-def chains of combinational logic, and inline assignments from the defining combinational groups.

Inline the assignment into the originGroup.

Things which stop recursive inlining (or in other words, what breaks combinational paths).

  • Component inputs
  • Register and memory reads
  • Constant ops (constant ops are not evaluated by any group)
  • Multiplication pipelines are sequential.
  • 'While' return values (these are registers, however, 'while' return values have at the current point of conversion not yet been rewritten to their register outputs, see comment in LateSSAReplacement)

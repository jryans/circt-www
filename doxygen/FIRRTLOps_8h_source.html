54 /// Compute the flow for a Value, \p val, as determined by the FIRRTL
55 /// specification. This recursively walks backwards from \p val to the
56 /// declaration. The resulting flow is a combination of the declaration flow
57 /// (output ports and instance inputs are sinks, registers and wires are
58 /// duplex, anything else is a source) and the number of intermediary flips.
59 /// An even number of flips will result in the same flow as the declaration.
60 /// An odd number of flips will result in reversed flow being returned. The
61 /// reverse of source is sink. The reverse of sink is source. The reverse of
62 /// duplex is duplex. The \p accumulatedFlow parameter sets the initial flow.
63 /// A user should normally \a not have to change this from its default of \p
64 /// Flow::Source.
65 Flow foldFlow(Value val, Flow accumulatedFlow = Flow::Source);
